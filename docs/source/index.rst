SpatioloJI: Spatial Analysis for Single-Cell Imaging Data -- Core Foundation for Ji Universe
========================================================

.. image:: _static/logo.svg
   :width: 300px
   :align: center

**SpatioloJI** is a Python package for analyzing spatial single-cell imaging data and is a core foundation for Ji Universe

Key Features
-----------

* Specialized for imaging-based spatial profiling technologies
* Comprehensive spatial visualization tools
* Spatial stats analysis
* Integration with standard single-cell analysis workflows of Scanpy
